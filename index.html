Talker

Advancet Exception Handler and Logger for dart and Flutter

Quick Start →

🍜 Handle Exceptions easy

Forget Spaghetti code in exceptions catching. Give this task to the Talker

🚀 Highly expandable

Talker API provides a lot of customization ways. You can configure everything different output, formatting, etc

📱 Logs at Display in runtime

Talker can show all logs on display in your Flutter application in runtime. It helps developing features.

🎨 Make Custom Logs

Implement your own console logs with custom colors, titles, messages, types. For examle. For example - separate type for http requests or state management. To make them stand out from the others

👁 Observable

All logs, errors and exceptions are observable. The project provides an opportunity to connect your listener and do your own logic when receiving messages.
